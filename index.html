JSON Placeholder Exercise

An exercise using JSON with the JavaScript Fetch API

Welcome

JSON

JSON is a commonly used file format, the website, JSON Placeholder, has an API available that we can use to send requests to. This frontend website already has the ability to create users and read users using this fake API, see:

  • scripts/users/create-user.js and html/create.html for creating users
  • scripts/users/read-user.js and html/read.html for reading users

See here for a guide on using the API

Your tasks:

  • Implement the update route for users, this is a PUT request (MEDIUM)
  • Implement the delete route for users, this is a DELETE request (EASY)
  • Implement the ability to read the posts of a user, for example: https://jsonplaceholder.typicode.com/users/1/posts returns a list of the users posts (HARD)
  • Implement the ability to read the todo items of a user, for example: https://jsonplaceholder.typicode.com/users/1/todos returns a list of the users todos (HARD)
  • Implement the ability to read the comments of a user on posts, for example: https://jsonplaceholder.typicode.com/users/1/comments returns a list of the users comments made on various posts (HARD)

TIP: Each route will require you to create a form, and JavaScript for processing that form (and possibly rendering data in the DOM)
